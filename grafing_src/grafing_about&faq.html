What is Grafing?

A graphing calculator... The best way you can explain. You can use it to plot points and functions. Plus inbuilt Linear Regression and Polygonal Regression algorithms are icing on the top. My initial plan was to build a mathematical engine that can perform different types of mathematical operations which does not require much of a lower level implementation. But I somehow ended up with a graphing tools.

Languages & libraries used to build this?

This web version uses HTML, CSS and JavaScript. Special thanks to these libraries - JSX Graph, Tabulator JS, MathJS, BootStrap, jQuery EasyUI. Everything is executed on the client side, on your browser.

Any caveat?

The equations plot does not come with WYSIWYG editor. It lacks a slider option for a coefficient of variable. But I'm working on a desktop app version where I will try my best to make it look more organic.

Can I fork this project?

You're welcome. Go to the repo, download the source files and start your own project.

Having problem to plot a function?

An error log appears whenever an exception is catched.
Here's a few things to check -

  • Symbolic coefficient can not be used. Like for y= m*x + c you must replce coefficients, m and c with some numerical values.
  • Use math. prefix whenevr using any square-root or trigionometric or logarrithemic functions. Example: Use math.sin(1/x) instead of sin(x). Instead of root(x) use math.sqrt(x).
  • Check if your parenthesis are closed properly.
