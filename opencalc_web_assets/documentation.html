Welcome to Project Documentation

Everything started when we got a long vacation during the 2020 COVID-19 pandemic. I thought this might be a great time to learn some new languages. I was planning to teach myself JavaScript for a while. So I just started with that. And probably the best way to learn a new programming language is to build something out of what you're teaching yourself! This project is still in development. So you might face some issues. You can contribute to the project by forking the source code. Please note that I neither a professional programmer nor from the computer science field. But I love making & breaking things and playing with computers. I wrote this in my free time. So the code might be written in a silly unprofessional way(lol).
