Project Documentation

Everything started when I got a long vacation from the university during the 2020 COVID-19 pandemic. I thought this might be a great time to learn some new languages. I was planning to teach myself JavaScript for a while. So I just started with that. And probably the best way of learning a new programming language is to build something out of what you're teaching yourself! This project is still in the development process. So you might face some issues. This is an open-source project and you can contribute to the project. Please note that I'm neither a professional programmer nor from the computer science field. But I love making & breaking things and playing with computers. I wrote this in my free time. So the code might be written in a silly unprofessional way. - Happy Coding!
